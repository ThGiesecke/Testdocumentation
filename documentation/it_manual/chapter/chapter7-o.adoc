=== Wartung und Pflege

==== Regelmäßige Wartungsaufgaben

Folgende Wartungsaufgaben sind in festen Intervallen durchzuführen:

- Wöchentliche Kontrolle der Systemprotokolle auf Fehler und Warnungen
- Monatliche Überprüfung der Backup-Jobs und Durchführung von Test-Rücksicherungen
- Einspielung von Sicherheitsupdates für Server-Betriebssystem, Laufzeitumgebungen und Drittkomponenten (nach Freigabe)
- Halbjährliche Kontrolle und Anpassung der Benutzer- und Rechteverwaltung
- Prüfung und ggf. Erneuerung von SSL-Zertifikaten sowie Lizenzschlüsseln
- Reinigung und Wartung der Serverhardware nach IT-Betriebsrichtlinie

==== Backup-/Restore-Anleitung

Backup-Erstellung (täglich, automatisiert):

- Datenbank-Backup:
  1. Öffnen Sie das Datenbank-Management-Tool (z. B. SQL Server Management Studio).
  2. Wählen Sie die Datenbank „kundenmanagerpro“ aus und starten Sie die Sicherung in das definierte Zielverzeichnis.
- Applikationsdaten sichern:
  1. Kopieren Sie das Applikationsverzeichnis in das Backup-Laufwerk mittels Script oder Backup-Software.

Backup-Überprüfung:

- Kontrollieren Sie nach jeder Sicherung das Logfile auf Fehlermeldungen.
- Monatlich wird ein Restore-Test in einer separaten Testumgebung durchgeführt.

Restore-Anleitung im Fehlerfall:

1. Auswahl der gewünschten Sicherung (Datum/Uhrzeit).
2. Rücksichern der Datenbank über das Datenbank-Management-Tool.
3. Zurücksichern der Applikationsdateien ins Zielverzeichnis.
4. Start der Anwendung und Systemprüfung.
5. Dokumentation des Restore-Vorgangs im Wartungsprotokoll.
